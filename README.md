# Hi there 👋 I'm Hanqi Wang

I'm a PhD student in Materials Science and Engineering at **Monash University**, working on computational materials science, magnetic topological materials, and low-energy electronic/spintronic materials.

My current research focuses on **first-principles design of 2D magnetic topological materials**, especially Janus V₂M(X,Y)₄ monolayers and bilayers, using DFT, spin–orbit coupling, Wannier tight-binding models, Berry curvature, Chern number calculations, and edge-state analysis.

## 🚀 Current project direction

My current PhD direction is to explore whether **Janus V₂M(X,Y)₄ bilayers** can realize mechanically switchable topological phases through interlayer sliding.

The long-term goal is to move from single-material studies toward **new family-level material prediction**, identifying new 2D magnetic systems with coupled electronic and magnonic topology.
